### Stackr — Never Solve the Same Error Twice
#### Stackr is a developer-focused knowledge base designed to help developers log, store, and reuse solutions to errors they encounter during development.

#### Instead of repeatedly searching for the same issues, developers can document errors once and quickly retrieve proven fixes.
### Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Node.js with TypeScript
Database: Firebase Realtime Database

##### Search functionality
Quickly find previously logged errors using keywords.
#### Tag-based filtering
Errors can be categorized using tags (e.g. node.js, react, database).
#### Log errors
Users can:
Enter error title
Describe the issue
Add tags
Provide a working solution
#### Persistent storage
All errors are stored in Firebase and can be retrieved anytime.

#### Architecture
Frontend (HTML/CSS/JS)
        ↓
Fetch API (HTTP requests)
        ↓
Node.js + Express (TypeScript backend)
        ↓
Firebase Realtime Database (JSON storage)
#### Use cases 
Can be used along AI tools, and can be beneficial as an inhouse tool
